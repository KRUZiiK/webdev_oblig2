Git & GitHub

Git er et versjonskontroll program som holder styr på flere versjoner av filer i noe du jobber med, gjerne kode men kan brukes til annet, så sant du går og oppdaterer hver gang du har gjort noe. Git gir også flere personer mulighet jobbe på samme prosjekt med samme filer og alltid ha tilgang til nyeste versjon gjennom samme versjonskontroll. samt mulighet til å se forskjeller i filer som mulig er endret av 2 personer for å gjøre sammenslåing enklere. Git programvare fungerer ved at du velger en mappe på pc-en hvor du enten oppretter eller henter inn et prosjekt. Om du hentet et eksisterende prosjekt kloner programmet alle filer til pc-en din, om ikke kan du bare start å jobbe fra mappen som vanlig. Etter hvert som du gjør endringer eller legger til nye filer kan du gjennom git skrive versjnons notater og pushe oppdateringene dine til hovedversjonen av prosjektet. Git kan brukes lokalt på en offline pc, men for mest utbytte kan brukes med online backing gjennom tjenester som github. Essensielt en cloud tjeneste hvor versjonene lagres istedenfor kun lokalt, som gjør det mulig å ha flere presjoner involvert i samme prosjekt…
